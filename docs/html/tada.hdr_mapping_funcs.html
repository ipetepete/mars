Functions intended to be referenced in Personality files. 
 
Each function accepts a dictionary of name/values (from a FITS header)
and returns a dictionary of new name/values.  Fields (names) may be
added but not removed in the new dictionary with respect to the
original.
 
In each function of this file:
  orig::    orginal header as a dictionary
  RETURNS:: dictionary that should be used to update the header
