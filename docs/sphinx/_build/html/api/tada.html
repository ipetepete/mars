tada.generate_files module¶

This file is ONLY as an aid to testing. Not used in production TADA.

The files generated will be valid FITS, but not valid to Archive Ingest. They should go through TADA, but fail on submit.

Each file is 5760 bytes. Took 14 seconds to create 10,000 files (on chimp16). Took 148 seconds to create 100,000 files (on chimp16).

tada.utils.log_traceback()¶

Log a traceback with sufficient detail to point to source of error. NOTE: The traceback (stack) itself is logged to INFO, not ERROR. This is to allow tests to use ERROR and WARNING logging to insure correct behavior. Stacks have line numbers so would cause unsability in GOLD files.
