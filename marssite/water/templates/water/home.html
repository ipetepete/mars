/schedule/2015/aug/23/ Get a days worth.
/schedule/today/ Get todays schedule.
/schedule/upload/ Upload XML file of schedule data and load it.

/schedule/update/2015B Update DMO telescope SEMESTER schedule using www.noao.edu./noaoprop svc.
http://www.noao.edu/noaoprop/schedule.mpl?semester=2015B The schedule from Dave Bell's DB (one semester)
/siap/ SIAP index of all files.
/siap/fquery?cols=reference&limit=2 Query voi.siap using form or URL parameter. Direct query using a string payload is available via: {% url 'siap:query_by_sql' %}
{% url 'siap:query_by_sql' %} Query voi.siap using SQL string.
For SQL in file use something like:
curl -X POST --data-binary @sql/files20.sql http://localhost:8000/siap/squery

For SQL in string use something like:
curl X POST --data-binary "select reference,dtacqnam from voi.siap limit 5" http://localhost:8000/siap/squery
{% url 'siap:query' %} Query voi.siap using web page.
/siap/tada/ List of all Archive files containing substring "TADA". Get different renderings based on requested content type. Examples: 
  curl -H "Content-type: application/json" 'http://localhost:8000/siap/tada/'
  curl -H "Content-type: text/csv" 'http://localhost:8000/siap/tada/' >x.csv
/siap/prop/PROPID Filenames associated with PROPID='2011A-0525'
{% url 'audit:submittal_list' %} For monitoring progress ingestion. Reconcile files POSTED (via dropbox or direct submit) verses INGESTED into the archive. Compare the list of files posted via raw_post from all domes to the list of files stored in the Noao Science Archive.
{% url 'audit:submittal_add' %}

Reconcile files POSTED verses INGESTED into the archive. Compare the list of files posted via raw_post from all domes to the list of files stored in the Noao Science Archive.

Example:

curl -H "Content-Type: application/json" -X POST -d '{"source":"xyz","archive":"xyz","status":"NA1", "metadata":"NA2"}' http://localhost:8000/audit/add

/provisional/ List of all files stored in the Archive on a provisional basis. These can be deleted as a batch or individiually through web-services.
/provisional/stuff Insert all files matching "TADA like" reference file into provisional list.
/provisional/delete/REF_FILENAME/ Delete a file from Archive and the provisional list.
/provisional/rollback Delete all files that are in provisional list from Archive and from the provisional list.
/provisional/add/{ref_filename}/?source={source_filename} Add a file to list of provisional files. Overwrites previous REF_FILENAME if it exists. can use as in: 
  fname=/data/scraped/mtn_raw/ct15m-echelle/chi141225.1273.fits
  curl "http://localhost:8000/provisional/add/ref5/?source=$fname"
/docs/ Web-service API automatically generated documentation.
